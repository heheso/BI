Solution (Method 2: Using Provided Dataset eg Sales_Data.csv)

==============================
1. Import the Dataset in Excel
==============================
Open Microsoft Excel
Go to:
Data → From Text/CSV
Select file:
Sales_Data.csv
Click Load
The dataset will appear in Excel.

2. Create the Pivot Table & Chart
=================================
Go to:
Insert → PivotChart → PivotChart & PivotTable
Select Table/Range: Sales_Data sheet
Where to place: New Worksheet
Click OK

Drag fields as follows:
Rows → Region
Columns → Product
Values → TotalPrice (Sum)
